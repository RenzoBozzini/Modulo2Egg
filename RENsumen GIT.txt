Para inicializar un control de revisiones de git y subirlo a github:

git bash here con click derecho en la carpeta con la que quiero trabajar con control de revisiones.

En consola:
$ git init
$ git remote add origin pEgArURL
$ git status
$ git remote -v
$ git add .
$ git commit -m "Comentario del contenido
$ git push origin master

Para actualizar dicha carpeta:
$ git add .
$ git commit -m "Comentario del contenido"
$ git push origin master
